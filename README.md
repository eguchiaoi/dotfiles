## VIM install
Install vim by homebrew
~~~~
$ brew install vim
~~~~

## Font
Install Hack from brew cask.
~~~~
$ brew cask install caskroom/fonts/font-hack-nerd-font
~~~~

Then, fix the "Hack Regular Nerd Font Complete 13pt." in profile of the terminal.
